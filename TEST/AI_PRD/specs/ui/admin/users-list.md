# 관리자 사용자 목록 화면 (Admin User List)

## 개요
관리자 - 사용자 목록 조회 및 관리

## 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│  [로고] Stock Keeper Admin          [관리자명] [로그아웃]    │
├─────────┬───────────────────────────────────────────────────┤
│         │                                                   │
│ 대시보드 │  사용자 관리                         [CSV 내보내기]│
│         │ ────────────────────────────────────────────────  │
│ 사용자 ◀│  [🔍 검색...]  [역할 ▼] [멤버십 ▼] [상태 ▼]        │
│         │                                                   │
│ 모니터링 │  ┌────────────────────────────────────────────┐  │
│         │  │ 이메일       │닉네임│역할│멤버십│상태│가입일│  │
│ 설정    │  ├────────────────────────────────────────────┤  │
│         │  │user@...     │홍길동│USER│ NOT │활성│01-01│  │
│         │  │admin@...    │관리자│ADMIN│ NOT │활성│01-01│  │

│         │  └────────────────────────────────────────────┘  │
│         │                                                   │

└─────────┴───────────────────────────────────────────────────┘
```

## 컴포넌트

### 1. 검색 바
- 이메일, 닉네임으로 검색
- 입력 후 Enter 또는 500ms debounce

### 2. 필터 드롭다운
| 필터 | 옵션 |
|------|------|
| 역할 | 전체, USER, ADMIN |
| 멤버십 | 전체, NOT, PRO |
| 상태 | 전체, 활성, 비활성 |

### 3. 사용자 테이블
| 컬럼 | 정렬 가능 | 클릭 동작 |
|------|----------|----------|
| 이메일 | X | 상세 모달 열기 |
| 닉네임 | O | - |
| 역할 | X | 배지 표시 |
| 멤버십 | X | 배지 표시 |
| 상태 | X | 활성/비활성 표시 |
| 가입일 | O | - |
| 마지막 접속 | O | - |

### 4. 페이지네이션
- 페이지 번호 표시
- 페이지당 건수: 20, 50, 100

### 5. CSV 내보내기 버튼
- 현재 필터 조건 기준 CSV 다운로드

## 상호작용

| 액션 | 동작 |
|------|------|
| 행 클릭 | 사용자 상세 모달 열기 |
| 검색 입력 | 500ms 후 자동 검색 |
| 필터 변경 | 즉시 재조회 |
| 정렬 클릭 | 해당 컬럼 기준 정렬 토글 |
