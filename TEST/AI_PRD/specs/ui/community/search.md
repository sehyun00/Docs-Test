# 검색 화면 (Search Screen)

## 개요
커뮤니티 통합 검색 (종목, 게시글, 사용자)

## 레이아웃

```
┌─────────────────────────────────┐
│  ← [🔍 검색어 입력...]          │
├─────────────────────────────────┤
│                                 │
│  종목                  더 보기 >│
│  ┌─────────────────────────────┐│
│  │ 삼성전자 (005930)    70,000 ││
│  │ 삼성SDI (006400)    450,000 ││
│  │ 삼성바이오 (207940) 780,000 ││
│  └─────────────────────────────┘│
│                                 │
│  게시글                더 보기 >│
│  ┌─────────────────────────────┐│
│  │ 삼성전자 실적 분석    ❤️ 42 ││
│  │ 삼성그룹 전망          ❤️ 28 ││
│  └─────────────────────────────┘│
│                                 │
│  사용자                더 보기 >│
│  ┌─────────────────────────────┐│
│  │ 🧑 삼성러버    팔로워 123   ││
│  │ 🧑 삼성맨      팔로워 45    ││
│  └─────────────────────────────┘│
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 검색 헤더
- 뒤로가기 (←)
- 검색 입력필드 (자동 포커스)
- 검색어 지우기 (X)

### 2. 종목 섹션
| 요소 | 설명 |
|------|------|
| 종목명 | 종목명 (종목코드) |
| 현재가 | 현재 주가 |
| 등락률 | +2.5% (빨강) / -1.2% (파랑) |
| 더 보기 | 종목 전체 결과 화면 |

### 3. 게시글 섹션
| 요소 | 설명 |
|------|------|
| 제목 | 게시글 제목 |
| 좋아요 | ❤️ N |
| 더 보기 | 게시글 전체 결과 화면 |

### 4. 사용자 섹션
| 요소 | 설명 |
|------|------|
| 프로필 | 캐릭터 아이콘 |
| 닉네임 | 사용자 닉네임 |
| 팔로워 | 팔로워 수 |
| 더 보기 | 사용자 전체 결과 화면 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 검색어 입력 | 500ms 디바운스 후 검색 |
| 종목 클릭 | 해당 종목 포함 포트폴리오 리스트 |
| 게시글 클릭 | 게시글 상세 화면 |
| 사용자 클릭 | 프로필 화면 |
| 더 보기 클릭 | 해당 카테고리 전체 결과 (페이지네이션) |

## 상태

| 상태 | UI |
|------|-----|
| 초기 | 검색어 입력 안내 문구 |

| 에러 | 에러 메시지 + 재시도 |
