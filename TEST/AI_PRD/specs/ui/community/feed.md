## 개요
커뮤니티 메인 피드 - 게시글/포트폴리오 통합 목록

## 컴포넌트

### 1. 헤더
- 타이틀: "커뮤니티"
- 검색 버튼 (🔍): 검색 화면 이동
- 작성 버튼 (➕): 게시글 작성 화면 이동

### 2. 필터 바
| 필터 | 옵션 |
|------|------|
| 카테고리 | 전체, 포트폴리오, 토론, 질문, 국내주식, 해외주식, 자유 |
| 정렬 | 최신순 (P2), 인기순 (P3), 팔로잉 (P3) |

### 3. 피드 카드
#### 게시글 카드
| 요소 | 설명 |
|------|------|
| 카테고리 배지 | 말머리 표시 |
| 제목 | 1줄, 말줄임 |
| 미리보기 | 2줄, 말줄임 |
| 이미지 수 | 🖼 N |
| 좋아요 수 | ❤️ N |
| 댓글 수 | 💬 N |
| 작성자 | 닉네임 · 시간 |

#### 포트폴리오 카드
| 요소 | 설명 |
|------|------|
| 아이콘 | 📊 포트폴리오 |
| 이름 | 포트폴리오명 |
| 정보 | 종목 N개 · 작성자 |
| 좋아요/복사 | ❤️ N  📋 N |

### 4. Pull to Refresh
- 당겨서 새로고침

### 5. 무한 스크롤
- 하단 도달 시 자동 로드

## 상호작용

| 액션 | 동작 |
|------|------|
| 카드 탭 | 상세 화면 이동 |
| 좋아요 탭 | 좋아요 토글 (본인인증 필요) |
| 작성자 탭 | 프로필 화면 이동 |
| Pull to Refresh | 최신 데이터 로드 |
| 스크롤 끝 | 다음 페이지 로드 |

## 본인인증 미완료 시

- 조회는 가능
- 좋아요, 댓글, 작성 클릭 시 → "본인인증이 필요합니다" 모달 → 인증 화면 이동

## 상태

| 상태 | UI |
|------|-----|
| 로딩 중 | 스켈레톤 카드 표시 |
| 빈 피드 | "아직 게시글이 없습니다" + 작성 유도 버튼 |
| 에러 | 에러 메시지 + 재시도 버튼 |
| 추가 로딩 | 하단 스피너 표시 |
