# 포트폴리오 목록 화면 (Portfolio List)

## 개요
사용자 포트폴리오 목록 표시 및 관리

│  │ ≡ 내 포트폴리오              ││
│  │   5개 종목 · ₩10,000,000    ││

## 컴포넌트

### 1. 헤더
- 타이틀: "포트폴리오"
- 추가 버튼 (➕): 새 포트폴리오 생성 모달

### 2. 포트폴리오 카드
| 요소 | 설명 |
|------|------|
| 드래그 핸들 | ≡ (롱프레스로 순서 변경) |
| 이름 | 포트폴리오 이름 (20자 초과 시 ...) |
| 종목 수 | n개 종목 |
| 총 평가금액 | ₩N,NNN,NNN |
| 수정일 | 상대 시간 |

### 3. 스와이프 액션
- 왼쪽 스와이프: 수정 버튼
- 오른쪽 스와이프: 삭제 버튼

## 상호작용

| 액션 | 동작 |
|------|------|
| 카드 탭 | 포트폴리오 상세 화면 |
| 추가 버튼 탭 | 생성 모달 표시 |
| 롱프레스 + 드래그 | 순서 변경 |
| 좌측 스와이프 → 수정 | 수정 모달 |
| 우측 스와이프 → 삭제 | 삭제 확인 다이얼로그 |
| Pull to Refresh | 목록 새로고침 |

## 상태

| 상태 | UI |
|------|-----|
| 로딩 중 | 스켈레톤 카드 |
| 빈 목록 | "포트폴리오를 추가해보세요" + 추가 버튼 |
| 에러 | 에러 메시지 + 재시도 버튼 |
| 5개 도달 | 추가 버튼 비활성화 |
