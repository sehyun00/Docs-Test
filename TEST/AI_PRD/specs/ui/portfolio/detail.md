# 포트폴리오 상세 화면 (Portfolio Detail)

## 개요
포트폴리오 종목 목록 및 리밸런싱 정보 표시

```

├─────────────────────────────────┤
│  ┌─────────────────────────────┐│
│  │  총 평가금액                 ││
│  │  ₩10,000,000               ││
│  │  현금 ₩1,000,000 · $500    ││

│  │ ⚠️ 리밸런싱 필요            ││
```

## 컴포넌트

### 1. 헤더
- 뒤로가기 (←)
- 포트폴리오 이름 + 설명
- 편집 버튼 (✏️)

### 2. 요약 카드
| 요소 | 설명 |
|------|------|
| 총 평가금액 | 종목 + 현금 합계 |
| 현금 | 원화 + 달러 표시 |

### 3. 리밸런싱 알림 카드
- 조건: 임계값 초과 종목 있을 때만 표시
- 버튼: 리밸런싱 화면 이동

### 4. 종목 카드
| 요소 | 설명 |
|------|------|
| 종목명 | 종목 이름 (코드) |
| 목표 비율 | 목표 N% |
| 현재가 × 수량 | 가격 × 보유량 |
| 평가금액 | 현재 비율 % |
| 진행바 | 현재 vs 목표 시각화 |
| 괴리율 | 현재 - 목표 (±N%p) |

## 상호작용

| 액션 | 동작 |
|------|------|
| 종목 카드 탭 | 종목 수정 바텀시트 |
| 종목 좌측 스와이프 | 삭제 버튼 |
| 종목 추가 버튼 | 종목 검색/추가 화면 |
| 리밸런싱 확인 버튼 | 리밸런싱 화면 |
| 편집 버튼 | 포트폴리오 수정 모달 |
| Pull to Refresh | 종목 현재가 새로고침 |

## 상태

| 상태 | UI |
|------|-----|
| 로딩 중 | 스켈레톤 |
| 종목 없음 | "종목을 추가해보세요" + 추가 버튼 |
| 에러 | 에러 메시지 + 재시도 버튼 |
| 현재가 업데이트 중 | 개별 종목 로딩 스피너 |
