# 로그인 화면 (Login Screen)

## 개요
Google OAuth 로그인 진입점

## 레이아웃

```
┌─────────────────────────────────┐
│                                 │
│                                 │
│                                 │
│         [앱 로고 이미지]          │
│                                 │
│       Stock Keeper              │
│    포트폴리오 리밸런싱 앱         │
│                                 │
│                                 │
│    ┌─────────────────────────┐  │
│    │  [G] Google로 시작하기    │  │
│    └─────────────────────────┘  │
│                                 │
│                                 │
│    로그인하면 서비스 이용약관 및   │
│    개인정보처리방침에 동의하게     │
│    됩니다.                       │
│                                 │
│    [이용약관] [개인정보처리방침]   │
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 앱 로고
- 크기: 120x120 px
- 위치: 화면 상단 1/3 지점

### 2. 앱 타이틀
| 요소 | 스타일 |
|------|--------|
| 메인 | "Stock Keeper", 24px, Bold, Primary |
| 서브 | "포트폴리오 리밸런싱 앱", 14px, Secondary |

### 3. Google 로그인 버튼
| 속성 | 값 |
|------|-----|
| 너비 | 화면 80% |
| 높이 | 48px |
| 배경 | 흰색 (Google 가이드라인) |
| 테두리 | 1px, #DADCE0 |
| 아이콘 | Google 로고 (왼쪽) |
| 텍스트 | "Google로 시작하기", 14px |

### 4. 약관 동의 문구
- 크기: 12px, Text Secondary
- 링크: 밑줄, 클릭 시 웹뷰 열기

## 상호작용

| 액션 | 동작 |
|------|------|
| 버튼 클릭 | Google OAuth 플로우 시작 |
| 로그인 성공 (신규) | 프로필 입력 화면 이동 |
| 로그인 성공 (기존) | 홈 화면 이동 |
| OAuth 취소 | 화면 유지 |
| OAuth 실패 | 에러 토스트 표시 |

## 상태

| 상태 | UI |
|------|-----|
| 기본 | 버튼 활성화 |
| 로딩 중 | 버튼에 스피너 표시, 버튼 비활성화 |
| 에러 | 토스트 메시지 표시 |

## 에러 처리

| 상황 | 메시지 |
|------|--------|
| 네트워크 오류 | "인터넷 연결을 확인해주세요" |
| OAuth 실패 | "로그인에 실패했습니다. 다시 시도해주세요" |
| 서버 오류 | "일시적인 오류가 발생했습니다" |
