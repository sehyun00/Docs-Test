# 종목 추가 화면 (Stock Add)

## 개요
종목 검색 및 포트폴리오에 추가

```

│  └─────────────────────────────┘│
│                                 │
│  검색 결과                      │
│  ┌─────────────────────────────┐│
```

## 컴포넌트

### 1. 검색 화면
| 요소 | 설명 |
|------|------|
| 검색 입력 | 종목명/코드 입력 (자동 포커스) |
| 검색 결과 | 종목명, 코드, 현재가, 등락률 |

### 2. 입력 화면
| 요소 | 설명 |
|------|------|
| 현재가 | 실시간 가격 + 등락률 |
| 보유 수량 | 숫자 입력 or +/- 버튼 |
| 목표 비율 | 슬라이더 + 직접 입력 |
| 남은 비율 | 100% - 기존 종목 합계 |
| 추가 버튼 | 유효성 통과 시 활성화 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 검색어 입력 | 500ms 디바운스 후 검색 |
| 검색 결과 탭 | 입력 화면으로 이동 |
| 수량 변경 | 평가금액 자동 계산 |
| 비율 슬라이더 | 실시간 비율 표시 |
| 추가 버튼 | 서버 저장 → 상세 화면 복귀 |

## 상태

| 상태 | UI |
|------|-----|
| 검색 중 | 스피너 |
| 검색 결과 없음 | "검색 결과가 없습니다" |
| 비율 초과 | 슬라이더 빨간색, 버튼 비활성화 |
| 저장 중 | 버튼 로딩 상태 |
