# 설정 페이지 시나리오 (Settings)

## 개요

- 앱 설정 및 사용자 정보 관리
- 알림 설정 (전체/포트폴리오별)
- 계정 관리 (로그아웃, 삭제)

## 진입 조건

| 조건 | 동작 |
|------|------|
| 홈 → 설정 아이콘 (⚙️) | 설정 화면 표시 |

## 설정 그룹 구조

```
설정
├─ 프로필
│   └─ 프로필 편집 → (닉네임, 투자 성향)
│
├─ 알림 설정
│   ├─ 푸시 알림 [토글]
│   ├─ 알림 주기 → 매일/매주/2주/한달
│   └─ 알림 시간 → 타임피커
│
├─ 앱 정보
│   ├─ 버전
│   ├─ 이용약관 → 웹뷰
│   ├─ 개인정보처리방침 → 웹뷰
│   └─ 오픈소스 라이선스 → 웹뷰
│
└─ 계정
    ├─ 로그아웃
    └─ 계정 삭제 → 2단계 확인
```

## 컴포넌트

### 프로필 카드

| 요소 | 설명 |
|------|------|
| 프로필 이미지 | Google 프로필 또는 이니셜 |
| 닉네임 | 사용자 닉네임 |
| 이메일 | Google 계정 이메일 |

### 알림 설정

| 항목 | 옵션 |
|------|------|
| 푸시 알림 | ON/OFF 토글 |
| 알림 주기 | 매일/매주/2주/한달 |
| 알림 시간 | 시간 선택 (기본 16:00) |

## 상호작용

| 액션 | 동작 |
|------|------|
| 프로필 카드 탭 | 프로필 편집 화면 |
| 알림 토글 | 알림 ON/OFF (OFF 시 하위 비활성화) |
| 알림 주기 탭 | 주기 선택 모달 |
| 알림 시간 탭 | 타임피커 |
| 약관 링크 탭 | 웹뷰 열기 |
| 로그아웃 탭 | 확인 팝업 → 로그인 화면 |
| 계정 삭제 탭 | 2단계 확인 → 로그인 화면 |

## 상태

| 상태 | UI |
|------|-----|
| 알림 ON | 모든 설정 활성화 |
| 알림 OFF | 하위 설정 회색 비활성화 |
| 알림 권한 거부 | "설정에서 알림을 허용해주세요" 안내 |

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 로그아웃 | 확인 팝업 "정말 로그아웃하시겠습니까?" |
| 계정 삭제 | 2단계: 이메일 입력 확인 |
| 알림 권한 거부됨 | 설정 앱으로 이동 안내 |
| 네트워크 오류 | 오프라인 표시 |
