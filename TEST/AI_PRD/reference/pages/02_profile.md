# 프로필 입력 페이지 시나리오 (Profile)

## 개요

- 신규 사용자 개인화 정보 수집 화면
- 닉네임 필수, 투자 성향 선택
- 완료 시 기본 포트폴리오 자동 생성

## 진입 조건

| 조건 | 동작 |
|------|------|
| 신규 로그인 (is_new_user=true) | 프로필 화면 표시 |
| 기존 사용자 | 스킵 → 홈 |

## 레이아웃

```
┌─────────────────────────────────────┐
│  ←  프로필 설정          [건너뛰기] │
├─────────────────────────────────────┤
│                                     │
│  [1. 로그인] ──●── [2. 프로필] ──○── [3. 시작] │
│                                     │
│  닉네임                             │
│  ┌─────────────────────────────┐   │
│  │ 투자왕김철수           8/10 │   │
│  └─────────────────────────────┘   │
│  2~10자 이내로 입력해주세요         │
│                                     │
│  투자 성향 (선택)                   │
│  [안정형] [중립형🔘] [공격형]       │
│                                     │
│  ┌─────────────────────────────┐   │
│  │       시작하기               │   │
│  └─────────────────────────────┘   │
│                                     │
│  💡 프로필은 설정에서 수정할 수 있어요 │
└─────────────────────────────────────┘
```

## 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| 진행 인디케이터 | 현재 단계 표시 (1.로그인 → 2.프로필 → 3.시작) |
| 닉네임 입력 필드 | 2~10자, 자동 포커스, 글자 수 표시 |
| 투자 성향 선택 | 안정형/중립형/공격형 (선택) |
| 시작하기 버튼 | 닉네임 입력 시 활성화 |
| 건너뛰기 버튼 | 기본 닉네임으로 진행 |

## 입력 필드 상세

| 속성 | 값 |
|------|-----|
| 최소 길이 | 2자 |
| 최대 길이 | 10자 |
| 자동 포커스 | Yes |
| 실시간 검사 | Yes (글자 수 표시) |
| Return 키 | "완료" → 시작하기 실행 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 닉네임 입력 | 실시간 유효성 검사 |
| 투자 성향 선택 | 토글 선택 |
| 시작하기 탭 | 저장 → 홈 화면 → 기본 포트폴리오 생성 |
| 건너뛰기 탭 | 기본 닉네임("사용자123")으로 홈 이동 |
| 뒤로가기 | 로그인 화면 (로그아웃 처리) |

## 상태

| 상태 | UI |
|------|-----|
| Default | 닉네임 빈 필드, 버튼 비활성화 |
| Valid | 닉네임 2자 이상, 버튼 활성화 |
| Error | 1자만 입력, 빨간 테두리 + 경고 메시지 |
| Loading | 버튼 "저장 중..." + 스피너 |

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 닉네임 미입력 | 버튼 비활성화 + "닉네임을 입력해주세요" |
| 닉네임 1자 | "2자 이상 입력해주세요" |
| 닉네임 11자 이상 | 입력 제한 (10자 컷) |
| 특수문자 | 허용 |
| 저장 실패 | 토스트 "저장에 실패했습니다" |

## 반응형

| 화면 크기 | 조정 |
|----------|------|
| 소형 (<360px) | 기본 레이아웃 |
| 키보드 올라옴 | 입력 필드가 키보드 위로 스크롤 |
