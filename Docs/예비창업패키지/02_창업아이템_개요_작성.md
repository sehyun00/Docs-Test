## 🔴 문제 인식 (Problem)

### 개발 동기 및 필요성

<!-- 해결하고자 하는 문제와 개발이 필요한 이유를 간결하게 작성 -->

**시장 현황 및 문제점:**

```
예시: 개인투자자의 70%가 수익률 관리에 어려움을 겪고 있으며, 
기존 리밸런싱 서비스는 고액 자산가 중심으로 월 수십만원의 비용 발생
```

**작성:**

- 한국예탁결제원(2024) 자료에 따르면, 국내 개인투자자 1,400만 명 중 약 78%가 포트폴리오 관리 없이 개별 종목 위주로 투자
- 기존 로보어드바이저 서비스는 최소 투자금 100만원 이상, 수수료 연 0.5~1% 수준으로 진입장벽이 높음

**문제 해결의 필요성:**

```
예시: 저비용으로 누구나 접근 가능한 AI 기반 리밸런싱 솔루션 필요
```

**작성:**

- 소액 투자자도 비용 부담 없이 포트폴리오를 체계적으로 관리할 수 있는 서비스 필요
- 초보 투자자가 쉽게 학습하고, 자신의 투자 전략을 검증할 수 있는 백테스팅 도구 부재
- 투자 경험을 공유하고 서로 배울 수 있는 커뮤니티 플랫폼의 필요성 증가

## 🔵 성장 전략 (Scale-up)

### 시장 진입 및 성과 창출 전략

**경쟁사 분석 요점:**

```
예시:
| 경쟁사 | 강점 | 약점 | 우리의 차별점 |
|--------|------|------|---------------|
| A사    |      |      |               |
```

**작성:**

| 경쟁사 | 강점 | 약점 | 우리의 차별점 |
|--------|------|------|---------------|
| 파운트 (Fount) | 삼성증권 연계, AI 자산배분 | 최소 투자금 100만원, 수수료 연 0.5% | 최소 투자금 없음, 커뮤니티 기반 학습 |
| 핀트 (Fint) | 국내 최대 로보어드바이저 | 월 구독료 부담, 초보자 학습 기능 부족 | 토큰 기반 무료 체험, 백테스팅 무료 |
| 에임 (AIM) | 글로벌 ETF, 세금 최적화 | 해외 투자 중심, 국내 주식 미지원 | 국내 주식 중심, 한글 커뮤니티 |
| 토스증권 스톡팜 | 높은 사용자 기반, 간편 UI | 리밸런싱 기능 없음 | AI 리밸런싱 + 백테스팅 기능 |

**목표 시장 진입 전략:**

```
예시: 20-30대 초보 투자자 대상 SNS 마케팅 및 투자 커뮤니티 협업
```

**작성:**

- 20-30대 초보 투자자 대상 SNS 마케팅
- 커뮤니티의 외부 공유 -> 링크를 누르면 자동으로 어플로 유도


**비즈니스 모델 (수익화 모델):**

```
예시: 
- 프리미엄(구독 모델): 월 9,900원
- 부가수익: 광고, 제휴 금융상품
```

**작성:**

- **토큰 기반 수익 모델**: 프리미엄 기능(AI 리밸런싱 추천, 상세 백테스팅 등) 사용 시 토큰 소모
- **토큰 획득 방법**: 유료 구매 또는 광고 시청으로 획득
- **부가수익**: 인앱 광고, 제휴 금융상품 연계

**투자유치(자금확보) 전략:**

```
예시: 
- 1년차: 정부지원금 6,000만원
- 2년차: 엔젤투자 5,000만원 
- 3년차: 시리즈A 5억원 목표
```

**작성:**

- **1년차 (2026)**: 예비창업패키지 정부지원금 최대 1억원
- **2년차 (2027)**: 초기창업패키지 추가 지원 + 엔젤투자 5,000만원 유치 목표
- **3년차 (2028)**: MAU 10만 달성 후 시리즈A 5억원 투자유치 목표
- **자체 수익**: 토큰 판매 및 광고 수익으로 운영비 충당
