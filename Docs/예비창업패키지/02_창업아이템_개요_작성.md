## 🔴 문제 인식 (Problem)

### 개발 동기 및 필요성

<!-- 해결하고자 하는 문제와 개발이 필요한 이유를 간결하게 작성 -->

**시장 현황 및 문제점:**

```
예시: 개인투자자의 70%가 수익률 관리에 어려움을 겪고 있으며, 
기존 리밸런싱 서비스는 고액 자산가 중심으로 월 수십만원의 비용 발생
```

**작성:**

- **MZ세대 및 로보어드바이저 시장 급성장**: 국내 RA 시장 30조 원 전망, MZ세대 투자자산 비중 27.6%로 확대
- **개인투자자 손실 심화**: 코스피 상승에도 불구 개인투자자 54.6% 손실(평균 -931만 원), 감정적 매매가 주원인
- **기존 서비스 한계**: 고액 자산가 위주의 로보어드바이저와 단순 시세 조회 앱 간의 양극화로 'Actionable Insight' 부재

**문제 해결의 필요성:**

```
예시: 저비용으로 누구나 접근 가능한 AI 기반 리밸런싱 솔루션 필요
```

**작성:**

- 소액 투자자도 비용 부담 없이 포트폴리오를 체계적으로 관리할 수 있는 서비스 필요
- 초보 투자자가 쉽게 학습하고, 자신의 투자 전략을 검증할 수 있는 백테스팅 도구 부재
- 투자 경험을 공유하고 서로 배울 수 있는 커뮤니티 플랫폼의 필요성 증가

## 🟢 실현 가능성 (Solution)

### 개발 및 구체화 계획

<!-- 협약기간 내 제품/서비스로 구체화하는 방법 -->

**최종 산출물 형태 및 수량:**

```
예시: 
- iOS/Android 모바일 앱 1개
- AI 리밸런싱 알고리즘 1건
- 시연용 프로토타입 1건
```

**작성:**

- **Spring Boot + Redis + React Native** 하이브리드 앱 개발
- **한국투자증권 Open API** 실시간 시세 연동 및 **리밸런싱 알고리즘** 내재화

**차별성 및 경쟁력 확보 전략:**

- **Actionable Insight**: 단순 조회가 아닌 '최적 매매 수량' 계산 및 매수/매도 타이밍 자동 알림
- **고성능 처리**: Redis 캐싱(TTL 30초)으로 API 호출 비용 최소화 및 응답 속도 혁신
- **커뮤니티 결합**: "내 포트폴리오 수익률" 배틀 등 소셜 기능으로 사용자 락인(Lock-in)

## 🔵 성장 전략 (Scale-up)

### 시장 진입 및 성과 창출 전략

**경쟁사 분석 요점:**

```
예시:
| 경쟁사 | 강점 | 약점 | 우리의 차별점 |
|--------|------|------|---------------|
| A사    |      |      |               |
```

**작성:**

| 경쟁사 | 강점 | 약점 | 우리의 차별점 |
|--------|------|------|---------------|
| **F사** (파운트) | 대형 금융사 제휴 | 고액 최소투자금, 높은 수수료 | **제로 진입장벽**, 누구나 즉시 시작 |
| **P사** (핀트) | 자율주행 투자 | 월 구독료 부담, 낮은 자유도 | **커스터마이징**, 내 입맛대로 리밸런싱 |
| **S사** (스톡이벤츠) | 직관적 UI | 단순 조회만 가능 (Action 부재) | **Actionable Insight**, 매수/매도 가이드 |

**목표 시장 진입 전략:**

- **Core 타겟**: 투자자산 비중이 높은(27.6%) 2030 '스마트 개미' 집중 공략
- **바이럴 마케팅**: "AI vs 존버" 비교 리포트로 커뮤니티(토스, 뽐뿌) 내 FOMO 자극 및 유입 유도

**비즈니스 모델 (수익화 모델):**

- **Freemium 구독 (Core)**: 월 4,900원의 저렴한 비용으로 수천만 원 손실 방어 가치 제공
- **Micro-transaction**: 광고 시청/미션 수행으로 토큰 획득 → 유료 기능 1회권 구매 (진입장벽 제거)

**투자유치(자금확보) 전략:**

- **1년차 (2026)**: 예비창업패키지 **1억원** 확보 (MVP 개발)
- **2년차 (2027)**: 초기창업패키지 + 엔젤투자 **3억원** (정식 런칭)
- **3년차 (2028)**: Series A **20억원** 유치 (글로벌 진출)
- **자체 수익**: 3년차 매출 25억원 달성으로 자생력 확보

```
예시:

```

**작성:**

- **팀원 1 (Backend)**: 컴퓨터공학 전공, 프로젝트 5건(백엔드/풀스택) 수행, DB 설게 및 AWS 구축 역량 보유, 수상 4회
