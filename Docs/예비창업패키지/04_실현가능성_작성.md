## 2-1. 창업 아이템의 개발 계획

### 🛠️ 아이디어 제품·서비스 구체화 계획

**시스템 아키텍처 및 핵심 기술:**
- **Frontend**: React Native (iOS/Android 하이브리드 앱), 직관적인 UX를 위한 디바운스 검색 및 실시간 차트
- **Backend**: Spring Boot (안정성), Redis (실시간 시세 30초 캐싱 및 Rate Limit 처리)
- **Data**: 한국투자증권 Open API 실시간 연동, 자체 구축 DB (Portfolio/Rebalancing Logic)

**단계별 개발 프로세스 (Phase 1 MVP):**
1.  **인증 및 보안 (Auth)**: Google OAuth 2.0 및 JWT 기반 세션 관리 (개인 금융 정보 보호 최우선)
2.  **핵심 기능 (Core)**: 포트폴리오 CRUD, 실시간 평가금액 계산, 목표 비중 대비 리밸런싱 알고리즘 구현
3.  **외부 연동 (Integration)**: 한국투자증권 API 연동을 통한 실시간 종목 시세 및 계좌 잔고 조회
4.  **알림 시스템 (Notification)**: FCM(Firebase) 기반 매수/매도 타이밍 푸시 알림

### ✨ 창업 아이템의 차별성 및 경쟁력 확보 전략

**기술적 독창성 (Technical Edge):**
- **초정밀 리밸런싱 알고리즘**: 단순 비율 유지가 아닌, 거래 비용(수수료/세금)을 고려한 최적 매매 수량 산출 로직 보유
- **고성능 시세 처리**: Redis 캐싱 전략(TTL 30초)을 도입하여 API 호출 비용 최소화 및 응답 속도 0.1초 이내 보장

**시장 경쟁력 (Business Edge):**
- **제로 진입장벽**: 기존 증권사 계좌 그대로 연동 (타사는 전용 계좌 개설 필수)
- **커뮤니티 드리븐**: 단순 유틸리티를 넘어 투자 전략을 공유하고 검증(백테스팅)받는 소셜 기능 결합
