## 4-1. 대표자 및 팀원 보유 역량

### 👤 대표자 역량

#### 관련 경력

```
예시:
- 00기업 데이터 분석팀 (3년)
- 00스타트업 개발팀장 (2년)
- 총 IT 분야 경력 5년
```

**작성:**

- 팀장 및 프로젝트 리드 경험 2회
  - Stock Management Service (SMS) 프로젝트 팀장 (2025.03 ~ 2025.06)
  - RUD 주식 리밸런싱 웹 서비스 Full-Stack 개발 (2024.09 ~ 2024.12)
  - 00대학교 동아리 연동 애플리케이션 Front-End 개발 (2023.06 ~ 2023.08)

#### 관련 학력/전공

```
예시:
- 00대학교 컴퓨터공학과 석사 졸업
- 관련 전공: AI/머신러닝, 데이터사이언스
```

**작성:**

- 00대학교 주전공: 신소재공학과 / 복수전공: 컴퓨터공학과 학사 졸업 예정 (2026.02)
- 관련 전공: 컴퓨터공학, 프로그래밍, 데이터 분석

#### 보유 기술력/자격

```
예시:
- Python, TensorFlow, PyTorch 숙련
- 정보처리기사 자격증 보유
- AI 관련 논문 2편 발표
```

**작성:**

- Programming: Python, Java, JavaScript, C, SQL
- Framework: React, React Native, Spring Boot, Flask
- AI/ML: PyTorch, TensorFlow, CUDA GPU 학습
- Server/DB: Node.js, Firebase, Docker, MySQL, MariaDB, AWS
- DevOps: Git, GitHub, Jira, Notion, Figma
- 전자정부 표준프레임워크 교육 이수 (한국지능정보사회진흥원, 2024.10)
- IoT 빅데이터 응용 교육과정 수료 (00대학교 로봇플랫폼연구소, 2024.09~12)

#### 수상 및 프로젝트 경험

```
예시:
- 00 해커톤 대상 수상 (2024년)
- 00 AI 경진대회 우수상 (2025년)
- 관련 프로젝트 5건 수행
```

**작성:**

- 동상 / SW 중심대학사업단 기업연계 프로젝트 (2025.06)
  → AI 기반 포트폴리오 리밸런싱 서비스 (TGNN, DDPG 딥러닝 모델 개발)
- 우수상 / IoT 빅데이터응용교육과정 성과물 경진대회 (2024.12)
  → OCR 기반 주식 리밸런싱 웹 서비스 개발
- 관련 프로젝트 총 3건 수행 (풀스택 2건, AI 모델 개발 1건)

#### 전업 가능 여부 및 투입 시간

```
예시:
- 전업 예정 (협약 시작 후 1개월 내 퇴사 예정)
- 주당 투입 시간: 50시간 이상
```

**작성:**

- 전업 가능 (2026.02 졸업 예정, 협약 즉시 전업 가능)
- 주당 투입 시간: 50시간 이상

## 💡 팀 구성 작성 TIP

1. **역량 = 아이템과의 연관성** 강조

   - 단순 경력 나열보다 "왜 이 사업을 할 수 있는가"를 증명
2. **팀의 완결성** 보여주기

   - 기술, 마케팅, 운영 등 필수 역할이 채워져 있는지
3. **협력사 = 실행력 증명**

   - 실제 협약이 완료된 파트너가 있으면 높은 점수
4. **예정 팀원도 구체적으로**

   - 어떤 조건의 인재를 어떻게 채용할 것인지 계획 제시
