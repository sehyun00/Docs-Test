> **우선순위**: P2  
> **마지막 수정**: 2026-01-04

---

## 📋 화면 개요

| 항목 | 내용 |
|-----|------|
| **화면명** | 설정 |
| **목적** | 앱 설정 및 사용자 정보 관리 |
| **진입 경로** | 홈 → 설정 아이콘 (⚙️) |
| **이동 가능 화면** | ← 홈, → 프로필 편집, → 각 설정 상세 |

### 사용자 스토리
>
> "사용자로서 나는 **알림 설정을 개인화**하고, 필요시 계정 정보를 관리하고 싶다."

## ⚠️ 엣지 케이스 및 에러 상태

| 상황 | 현재 처리 | 권장 처리 |
|-----|----------|----------|
| 로그아웃 확인 | ❌ 없음 | 확인 팝업 |
| 계정 삭제 | ❌ 없음 | 2단계 확인 (비밀번호 재입력) |
| 알림 권한 거부됨 | ❌ 없음 | "설정에서 알림을 허용해주세요" 안내 |
| 네트워크 오류 | ❌ 없음 | 오프라인 표시 |

### 로그아웃 확인 팝업

```
┌─────────────────────────────────────┐
│                                     │
│   정말 로그아웃하시겠습니까?        │
│                                     │
│   데이터는 안전하게 보관됩니다.     │
│                                     │
│     [취소]         [로그아웃]       │
│                                     │
└─────────────────────────────────────┘
```

### 계정 삭제 확인 (2단계)

```
[1단계]
"정말 계정을 삭제하시겠습니까?
모든 포트폴리오 데이터가 영구 삭제됩니다."

[2단계]
"마지막 확인입니다.
이메일을 입력하여 계정 삭제를 완료하세요."
[investor@email.com]
        [계정 삭제]
```
