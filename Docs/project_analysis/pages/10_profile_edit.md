## 📋 화면 개요

| 항목 | 내용 |
|-----|------|
| **화면명** | 프로필 편집 |
| **목적** | 사용자 프로필 정보 수정 |
| **진입 경로** | 설정 → 프로필 항목 탭 |
| **이동 가능 화면** | ← 설정 |

### 사용자 스토리

> "투자자로서 나는 **닉네임**을 변경하여 포트폴리오에 표시되는 이름을 개인화하고 싶다."

## 🧩 구현 요소

### 화면 레이아웃

```
┌─────────────────────────────────────┐
│  ←  프로필 편집              [저장] │
├─────────────────────────────────────┤
│                                     │
│         ┌─────────┐                 │
│         │   김    │  ← 아바타       │
│         └─────────┘                 │
│                                     │
├─────────────────────────────────────┤
│ 닉네임                              │
│ ┌─────────────────────────────────┐ │
│ │ 투자왕김철수                    │ │
│ └─────────────────────────────────┘ │
│ 1~20자 이내로 입력해주세요          │
│                                     │
├─────────────────────────────────────┤
│ 이메일 (변경 불가)                  │
│ ┌─────────────────────────────────┐ │
│ │ investor@email.com     🔒       │ │
│ └─────────────────────────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

### 구현 기능

- ✅ 뒤로가기 버튼 (설정으로 복귀)
- ✅ 헤더에 저장 버튼
- ✅ 닉네임 입력 필드 (1~20자 검증)
- ✅ 이메일 읽기 전용 표시
- ✅ 아바타 (닉네임 첫 글자 자동 표시)
