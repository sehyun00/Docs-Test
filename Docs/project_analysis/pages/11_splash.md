## 📋 화면 개요

| 항목 | 내용 |
|-----|------|
| **화면명** | 스플래시 스크린 |
| **목적** | 앱 시작 시 브랜드 노출 및 로딩 표시 |
| **진입 경로** | 앱 실행 (제어판 🚀 버튼) |
| **이동 가능 화면** | → 온보딩 (첫 방문) / 로그인 (재방문) |

### 사용자 스토리

> "사용자로서 나는 앱이 시작될 때 **브랜드를 인지**하고 로딩 중임을 알고 싶다."

## 🎛️ 프로토타입 제어

### 제어판 버튼

| 버튼 | 기능 |
|------|------|
| 🚀 앱 시작 | 꺼진 화면 → 스플래시 → 온보딩/로그인 |
| 🔄 초기화 | 첫 방문 상태로 리셋 |
| S | 스플래시 화면 직접 이동 |

### 첫 방문 플래그

- `isFirstVisit = true`: 스플래시 후 **온보딩** 표시
- `isFirstVisit = false`: 스플래시 후 **로그인** 직행
