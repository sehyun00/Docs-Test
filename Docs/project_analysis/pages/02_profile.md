## 📋 화면 개요

| 항목 | 내용 |
|-----|------|
| **화면명** | 프로필 설정 |
| **목적** | 신규 사용자 개인화 정보 수집 |
| **진입 경로** | 로그인 화면 (신규 가입 시) |

## 💡 개선 아이디어

### 1. 프로필 사진 간소화

| 옵션 | 설명 | 권장 |
|-----|------|------|
| A) 제거 | MVP에서 프로필 사진 완전 제거 | ✅ 권장 |
| B) Google 자동 연동 | Google 프로필 이미지 자동 사용 | ⭐ 차선책 |
| C) 현재 유지 | 선택적 업로드 | ❌ 불필요한 마찰 |

### 2. 투자 성향 활용 방안

현재 수집만 하고 **활용처가 불분명**합니다.

**제안 1: 리밸런싱 임계값 기본값 설정**

| 성향 | 기본 임계값 | 설명 |
|-----|-----------|------|
| 안정형 | ±3% | 작은 괴리도 민감하게 감지 |
| 중립형 | ±5% | 권장 기본값 |
| 공격형 | ±10% | 느슨하게 관리 |

**제안 2: 수집 안 함 (MVP)**

- Phase 1에서는 투자 성향 선택 제거
- Phase 2에서 AI 추천 기능과 함께 도입

### 3. 진행 인디케이터 추가

```
[1. 로그인] ──●── [2. 프로필] ────○── [3. 시작]
```

- 현재 위치를 알려줌
- "거의 다 됐어요" 느낌 제공

### 4. 건너뛰기 옵션

```
┌─────────────────────────────────────┐
│  ←  프로필 설정          [건너뛰기] │
│     ...                             │
```

- 급한 사용자를 위한 탈출구
- 기본 닉네임: "사용자" + 임의 숫자

## ⚠️ 엣지 케이스 및 에러 상태

| 상황 | 현재 처리 | 권장 처리 |
|-----|----------|----------|
| 닉네임 미입력 | ❌ 없음 | 버튼 비활성화 + 힌트 "닉네임을 입력해주세요" |
| 닉네임 1자 | ❌ 없음 | 실시간 경고 "2자 이상 입력해주세요" |
| 닉네임 11자 이상 | ❌ 없음 | 입력 제한 (10자 컷) |
| 중복 닉네임 | ❌ 없음 | (중복 허용 권장 - 유저 간 상호작용 없음) |
| 특수문자 | ❌ 없음 | 허용 또는 필터링 정책 결정 필요 |

## 🔄 화면 상태 (States)

### 1. Default State (초기)

```
닉네임: [빈 필드] 
시작하기: [비활성화]
```

### 2. Valid State

```
닉네임: [투자왕김철수] ✓
시작하기: [활성화]
```

### 3. Error State

```
닉네임: [김] (빨간 테두리)
"2자 이상 입력해주세요"
시작하기: [비활성화]
```

### 4. Loading State

```
시작하기: [저장 중...  ⟳]
```

## 📱 입력 필드 UX

### 닉네임 필드 상세

| 속성 | 값 |
|-----|-----|
| 타입 | text |
| 최소 길이 | 2 |
| 최대 길이 | 10 |
| 플레이스홀더 | "사용할 닉네임을 입력하세요" |
| 자동 포커스 | ✅ Yes |
| 키보드 | 한/영 기본 |
| Return 키 | "완료" → 시작하기 실행 |

### 실시간 유효성 검사

```
입력 중:  [투자왕김]     8/10
                        ↑ 글자수 표시
```

## 📐 반응형 고려사항

| 화면 크기 | 조정 사항 |
|----------|----------|
| 소형 (< 360px) | 프로필 사진 영역 축소 또는 제거 |
