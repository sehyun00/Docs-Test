# 온보딩 페이지 시나리오 (Onboarding)

## 개요

- 앱 최초 실행 시 1회 노출되는 소개 화면
- 4개 슬라이드로 앱 핵심 가치 전달
- 완료/건너뛰기 시 로그인 화면으로 이동

## 진입 조건

| 조건 | 동작 |
|------|------|
| 최초 실행 (isFirstVisit=false) | 온보딩 표시 |
| 재실행 (isFirstVisit=true) | 스킵 → 로그인 |
| 앱 삭제 후 재설치 | 온보딩 다시 표시 |

## 슬라이드 구성

### Slide 1: 핵심 가치

```
┌─────────────────────────────────────┐
│              📊                     │
│       왜 리밸런싱이 중요할까요?     │
│   목표 비율을 유지하면 리스크를     │
│   줄이고 수익을 안정화할 수 있어요  │
│           ● ○ ○ ○                   │
│                        [건너뛰기 →] │
└─────────────────────────────────────┘
```

### Slide 2: 포트폴리오 생성

```
┌─────────────────────────────────────┐
│           [일러스트]                │
│     나만의 포트폴리오를 만들어요    │
│   + 버튼을 눌러 새 포트폴리오를     │
│        쉽게 추가할 수 있어요        │
│           ○ ● ○ ○                   │
│     [← 이전]          [건너뛰기 →] │
└─────────────────────────────────────┘
```

### Slide 3: 종목 추가

```
┌─────────────────────────────────────┐
│           [일러스트]                │
│   종목을 추가하고 목표 비중을 설정  │
│   원하는 비율만 입력하면            │
│   리밸런싱은 자동으로 계산해줘요    │
│           ○ ○ ● ○                   │
│     [← 이전]          [건너뛰기 →] │
└─────────────────────────────────────┘
```

### Slide 4: 시작하기

```
┌─────────────────────────────────────┐
│           [일러스트]                │
│       스마트한 투자 관리, 시작!     │
│   임계값 초과 시 알림을 받고        │
│      언제든 리밸런싱하세요          │
│           ○ ○ ○ ●                   │
│     [← 이전]           [시작하기]  │
└─────────────────────────────────────┘
```

## 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| 슬라이드 컨테이너 | Carousel (스와이프 가능) |
| 도트 인디케이터 | 현재 위치 표시 (클릭 이동) |
| 건너뛰기 버튼 | 즉시 로그인 화면 이동 |
| 시작하기 버튼 | Slide 4에서만 표시 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 좌/우 스와이프 | 슬라이드 이동 |
| 도트 인디케이터 탭 | 해당 슬라이드로 이동 |
| 건너뛰기 탭 | 로그인 화면으로 즉시 이동 |
| 시작하기 탭 | 로그인 화면으로 이동 |
| 이전 버튼 탭 | 이전 슬라이드 |

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 중간에 앱 종료 | 다음 실행 시 처음부터 |
| 네트워크 오류 | 영향 없음 (로컬 UI) |

## 로컬 스토리지

| Key | Type | 설명 |
|-----|------|------|
| isFirstVisit | boolean | 완료/건너뛰기 시 true로 설정 |
