# 프로필 편집 페이지 시나리오 (Profile Edit)

## 개요

- 사용자 프로필 정보 수정
- 닉네임 변경 (1~20자)
- 이메일은 읽기 전용

## 진입 조건

| 조건 | 동작 |
|------|------|
| 설정 → 프로필 항목 탭 | 프로필 편집 화면 표시 |

## 레이아웃

```
┌─────────────────────────────────────┐
│  ←  프로필 편집              [저장] │
├─────────────────────────────────────┤
│                                     │
│         ┌─────────┐                 │
│         │   김    │  ← 아바타       │
│         └─────────┘                 │
│                                     │
├─────────────────────────────────────┤
│ 닉네임                              │
│ ┌─────────────────────────────────┐ │
│ │ 투자왕김철수              12/20 │ │
│ └─────────────────────────────────┘ │
│ 1~20자 이내로 입력해주세요          │
│                                     │
├─────────────────────────────────────┤
│ 이메일 (변경 불가)                  │
│ ┌─────────────────────────────────┐ │
│ │ investor@email.com     🔒       │ │
│ └─────────────────────────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

## 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| 아바타 | 닉네임 첫 글자 자동 표시 |
| 닉네임 입력 필드 | 1~20자, 글자 수 카운트 |
| 이메일 필드 | 읽기 전용, 🔒 아이콘 |
| 저장 버튼 | 헤더 우측 |

## 입력 필드 상세

| 속성 | 값 |
|------|-----|
| 최소 길이 | 1자 |
| 최대 길이 | 20자 |
| 실시간 검사 | Yes (글자 수 카운트) |

## 상호작용

| 액션 | 동작 |
|------|------|
| 닉네임 입력 | 실시간 글자 수 표시 |
| 저장 버튼 | API 호출 → 성공 토스트 → 설정 복귀 |
| 뒤로가기 | 설정 화면 복귀 |

## 상태

| 상태 | UI |
|------|-----|
| 유효 | 저장 버튼 활성화 |
| 빈 값 | 저장 버튼 비활성화, 에러 표시 |
| 저장 중 | 저장 버튼 로딩 |

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 닉네임 빈 값 | 저장 버튼 비활성화 |
| 닉네임 20자 초과 | 입력 제한 |
| 저장 실패 | 에러 토스트 표시 |
