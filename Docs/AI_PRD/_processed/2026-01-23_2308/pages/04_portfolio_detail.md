# 포트폴리오 상세 페이지 시나리오 (Portfolio Detail)

## 개요

- 개별 포트폴리오의 종목별 현황 및 비율 확인
- 현재 비율 vs 목표 비율 비교
- 리밸런싱 필요 여부 판단

## 진입 조건

| 조건 | 동작 |
|------|------|
| 홈 → 포트폴리오 카드 탭 | 상세 화면 표시 |

## 컴포넌트

### Summary Grid

| 요소 | 설명 |
|------|------|
| 현금 비중 | 금액 + 비율 |
| 총 매입금 | 원금 합계 |
| 수익금 | 평가금액 - 매입금액 |
| 수익률 | 백분율 |

### 리밸런싱 인사이트 배너

| 요소 | 설명 |
|------|------|
| 상태 아이콘 | ⚠️ 필요 / ✅ 적정 |
| 초과/미달 종목 | 상위 2개 표시 |
| 분석 버튼 | 리밸런싱 화면 이동 |

### 종목 카드

| 요소 | 설명 |
|------|------|
| 종목명 + 코드 | 좌측 |
| 평가금액 | 우측 상단 |
| 수익률 | 우측 |
| 비율 바 | 현재 vs 목표 시각화 |
| 괴리 배지 | 초과/미달/적정 |

### 헤더 메뉴 (≡)

| 항목 | 동작 |
|------|------|
| 알림 설정 | 포트폴리오별 알림 ON/OFF 모달 |
| 편집 | 편집 모드 진입 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 종목 카드 탭 | 종목 상세 모달 (수량, 매입가 등) |
| 헤더 메뉴 (≡) | 알림 설정 / 편집 |
| 편집 모드 | 종목 삭제 버튼 (×) 노출 |
| 분석 버튼 | 리밸런싱 화면 이동 |
| + 종목 추가 | 종목 검색 화면 이동 |

## 상태

| 상태 | UI |
|------|-----|
| Loading | 스켈레톤 (Summary Grid + 종목 카드 3개) |
| Empty | "아직 종목이 없어요" + 종목 추가 버튼 |
| Editing | 헤더 "완료" 버튼, 종목별 삭제(×) 버튼 |
| Error | "데이터를 불러올 수 없습니다" + 재시도 |

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 종목 0개 | Empty State UI |
| 종목 1개 | 리밸런싱 불가 안내 |
| 비율 합계 ≠ 100% | 경고 표시 |
| 시세 조회 실패 | 마지막 캐시 + 오류 표시 |
| 상장폐지 종목 | 경고 배지 + 삭제 유도 |
