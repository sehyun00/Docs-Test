# 스플래시 페이지 시나리오 (Splash)

## 개요

- 앱 시작 시 브랜드 노출 및 로딩 표시
- 2초 후 자동 전환
- 첫 방문/재방문에 따라 다음 화면 분기

## 진입 조건

| 조건 | 동작 |
|------|------|
| 앱 실행 | 스플래시 표시 |

## 레이아웃

```
┌─────────────────────────────────────┐
│                                     │
│                                     │
│               📊                    │
│                                     │
│          Stock Keeper               │
│                                     │
│       ━━━━━━━━━━━━━━━━━            │
│         불러오는 중...               │
│                                     │
│                                     │
└─────────────────────────────────────┘
```

## 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| 로고 아이콘 | 📊 or 커스텀 아이콘, 펄스 애니메이션 |
| 앱 이름 | "Stock Keeper" |
| 프로그레스 바 | 로딩 애니메이션 |
| 로딩 텍스트 | "불러오는 중..." |

## 자동 전환 로직

```
스플래시 (2초)
    ↓
├─ isFirstVisit = true → 온보딩
└─ isFirstVisit = false → 로그인
```

## 타이밍

| 항목 | 값 |
|------|-----|
| 최소 표시 시간 | 2초 |
| 로고 애니메이션 | 펄스 (무한 반복) |
| 프로그레스 바 | 0% → 100% (2초) |

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 네트워크 오류 | 영향 없음 (로컬 UI) |
| 빠른 디바이스 | 최소 2초 대기 |
