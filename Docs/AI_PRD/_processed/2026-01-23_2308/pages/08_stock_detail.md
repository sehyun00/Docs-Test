# 종목 상세 페이지 시나리오 (Stock Detail)

> **Phase**: 2~3 (P2 기본 UI, P3 차트/뉴스)

## 개요

- 개별 종목의 상세 정보 및 차트 확인
- 보유 현황, 비율 현황, 수정/삭제 기능
- Page 방식 권장 (확장성 확보)

## 진입 조건

| 조건 | 동작 |
|------|------|
| 포트폴리오 상세 → 종목 카드 탭 | 종목 상세 화면 표시 |

## 컴포넌트

### 현재가 섹션

| 요소 | 설명 |
|------|------|
| 현재가 | ₩금액 |
| 등락 | +/-금액 (+/-%) |
| 기준 시간 | HH:MM 기준 |

### 보유 현황 섹션

| 요소 | 설명 |
|------|------|
| 보유 수량 | N주 |
| 평균 매입가 | ₩금액 |
| 평가금액 | ₩금액 |
| 수익금 | +/-₩금액 |
| 수익률 | +/-N.N% |

### 비율 현황 섹션

| 요소 | 설명 |
|------|------|
| 현재 비율 | N% + 바 시각화 |
| 목표 비율 | N% |
| 괴리 | +/-N% (임계값 초과/적정) |

## 상호작용

| 액션 | 동작 |
|------|------|
| ✏️ 아이콘 | 수량/목표비율 편집 모달 |
| 차트 기간 선택 | 1일/1주/1달/3달 차트 전환 |
| 수량 수정 버튼 | 수량 편집 모달 |
| 삭제 버튼 | 삭제 확인 팝업 |

## 엣지 케이스

| 상황 | 처리 |
|------|------|
| 시세 조회 실패 | 마지막 캐시 + "업데이트 실패" 표시 |
| 상장폐지 종목 | 경고 배너 + 삭제 유도 |
| 차트 데이터 없음 | "데이터 준비 중" 표시 |
