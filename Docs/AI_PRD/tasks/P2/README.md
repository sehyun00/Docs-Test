# Phase 2: 커뮤니티 기초 기능

## 🎯 목표

사용자 간 **포트폴리오 공유 및 소통**을 위한 커뮤니티 기반 구축

> [!NOTE]
> Phase 2는 Phase 1(MVP)이 완료된 후 시작합니다. 인증과 포트폴리오 기능이 안정적으로 동작해야 합니다.

## ⚠️ 주의사항

- **본인인증 필수 기능**: 게시글/댓글 작성은 본인인증 완료 사용자만 가능
- **포트폴리오 공개 설정**: P1의 포트폴리오에 공개 여부 필드 추가 필요
- **이미지 업로드**: S3 연동 필요 (게시글 이미지 첨부)
