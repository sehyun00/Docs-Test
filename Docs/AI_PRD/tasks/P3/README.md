# Phase 3: 커뮤니티 확장 기능

## 🎯 목표

커뮤니티의 **참여도와 재방문율**을 높이기 위한 소셜 및 게이미피케이션 기능 확장

> [!NOTE]

## ⚠️ 주의사항

- **대댓글**: 기존 댓글 시스템 확장, `parent_id` 계층 구조
- **팔로우**: 팔로워/팔로잉 카운트 캐시 관리 필요
- **랭킹**: 주간/월간 배치 작업 필요 (스케줄러)
- **배지**: 획득 조건 트리거 로직 복잡도 주의
