# 리밸런싱 분석 페이지 시나리오 (Rebalancing)

## 개요

- 목표 비율 달성을 위한 매수/매도 가이드 제공
- 현재 vs 목표 비율 비교 (도넛 차트)
- 구체적인 매수/매도 수량과 금액 제시

## 진입 조건

| 조건 | 동작 |
|------|------|
| 포트폴리오 상세 → 리밸런싱 분석 버튼 | 리밸런싱 화면 표시 |

## 컴포넌트

### 비교 차트

| 요소 | 설명 |
|------|------|
| 현재 도넛 | 현재 비율 시각화 |
| 목표 도넛 | 목표 비율 시각화 |
| 범례 | 종목별 색상 표시 |

### 액션 카드

| 요소 | 설명 |
|------|------|
| 액션 타입 | 🔴 매수 / 🔵 매도 |
| 종목명 | 우측 상단 |
| 비율 변화 | 현재 % → 목표 % |
| 권장 수량 | +N주 / -N주 |
| 예상 금액 | ₩금액 |

### 임계값 설정 (⚙️)

| 옵션 | 설명 |
|------|------|
| ±3% | 민감하게 관리 |
| ±5% | 권장 (기본값) |
| ±10% | 느슨하게 관리 |

## 상호작용

| 액션 | 동작 |
|------|------|
| ⚙️ 아이콘 탭 | 임계값 설정 모달 |
| 액션 카드 탭 | 상세 정보 + 복사 버튼 |
| [복사] 버튼 | 거래 정보 클립보드 복사 |
