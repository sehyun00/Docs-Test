# 계좌 생성/수정 모달 (Account Create/Edit)

## 개요

새 계좌 등록 및 기존 계좌 수정

## 레이아웃

```
┌─────────────────────────────────┐
│  새 계좌 등록               [X] │
├─────────────────────────────────┤
│                                 │
│  증권사                        │
│  ┌─────────────────────────────┐│
│  │ 한국투자증권              ▼ ││
│  └─────────────────────────────┘│
│                                 │
│  계좌번호 (선택)               │
│  ┌─────────────────────────────┐│
│  │ 12345678-01234567           ││
│  └─────────────────────────────┘│
│                                 │
│  [취소]           [등록하기]    │
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 증권사 선택

| 요소 | 설명 |
|------|------|
| 드롭다운 | 증권사 목록 선택 |
| 옵션 | 한투, 삼성, 키움, NH, 미래에셋 등 |

### 2. 계좌번호 입력

| 요소 | 설명 |
|------|------|
| 입력창 | 계좌번호 (선택 입력) |
| 암호화 | 저장 시 AES-256 암호화 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 등록하기 | API 호출 → 성공 시 목록 갱신 |
| 취소/닫기 | 모달 닫기 |

## 상태

| 상태 | UI |
|------|-----|
| 로딩 중 | 버튼 스피너 |
| 에러 | 에러 토스트 |
