# 계좌 목록 화면 (Account List)

## 개요

사용자 증권 계좌 목록 표시 및 관리

## 레이아웃

```
┌─────────────────────────────────┐
│  계좌 관리                  [➕] │
├─────────────────────────────────┤
│                                 │
│  ┌─────────────────────────────┐│
│  │ 한국투자증권                 ││
│  │ ****-****-1234              ││
│  │ 현금 ₩5,000,000             ││
│  │ 종목 5개                    ││
│  └─────────────────────────────┘│
│                                 │
│  ┌─────────────────────────────┐│
│  │ 삼성증권                    ││
│  │ ****-****-5678              ││
│  │ 현금 $1,000                 ││
│  │ 종목 3개                    ││
│  └─────────────────────────────┘│
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 헤더

- 타이틀: "계좌 관리"
- 추가 버튼 (➕): 새 계좌 생성 모달

### 2. 계좌 카드

| 요소 | 설명 |
|------|------|
| 증권사명 | 증권사 이름 |
| 계좌번호 | 마스킹 처리된 계좌번호 |
| 현금 정보 | 원화/달러 잔고 |
| 종목 수 | 보유 종목 개수 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 카드 탭 | 계좌 상세 화면 |
| 추가 버튼 탭 | 생성 모달 표시 |
| 좌측 스와이프 | 수정, 삭제 버튼 |

## 상태

| 상태 | UI |
|------|-----|
| 로딩 중 | 스켈레톤 |
| 빈 목록 | "계좌를 추가해보세요" |
| 에러 | 에러 메시지 + 재시도 |
