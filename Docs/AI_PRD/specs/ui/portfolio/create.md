# 포트폴리오 생성 모달 (Portfolio Create)

## 개요

새 포트폴리오 생성 화면 (모달/바텀시트)

## 레이아웃

```
┌─────────────────────────────────┐
│  새 포트폴리오 만들기       [X]  │
├─────────────────────────────────┤
│                                 │
│  이름                           │
│  ┌─────────────────────────────┐│
│  │ 배당주 포트폴리오            ││
│  └─────────────────────────────┘│
│  0/20자                         │
│                                 │
│  설명 (선택)                    │
│  ┌─────────────────────────────┐│
│  │ 고배당 종목 모음             ││
│  └─────────────────────────────┘│
│  0/100자                        │
│                                 │
│  [취소]           [만들기]      │
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 헤더

- 타이틀: "새 포트폴리오 만들기"
- 닫기 버튼 (X)

### 2. 이름 입력

| 요소 | 설명 |
|------|------|
| 라벨 | "이름" |
| 입력창 | 1~20자, 특수문자 제한 |
| 글자 수 | N/20자 |

### 3. 설명 입력

| 요소 | 설명 |
|------|------|
| 라벨 | "설명 (선택)" |
| 입력창 | 0~100자 |
| 글자 수 | N/100자 |

### 4. 버튼

- 취소: 모달 닫기
- 만들기: 포트폴리오 생성

## 상호작용

| 액션 | 동작 |
|------|------|
| 이름 입력 | 실시간 글자 수 표시 |
| 만들기 버튼 | 생성 API 호출 → 성공 시 목록 갱신 |
| 취소/닫기 | 모달 닫기 |

## 상태

| 상태 | UI |
|------|-----|
| 로딩 중 | 버튼 스피너 |
| 이름 미입력 | 만들기 버튼 비활성화 |
| 5개 초과 | 에러 메시지 표시 |
| 에러 | 에러 토스트 표시 |
