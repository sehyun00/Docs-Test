# 프로필 편집 화면 (Profile Edit)

## 개요

설정에서 진입하는 프로필 정보 편집 화면

## 레이아웃

```
┌─────────────────────────────────┐
│  ← 프로필 편집                   │
│                                 │
│         ┌─────────┐             │
│         │  📷    │             │
│         │ 프로필  │             │
│         └─────────┘             │
│        [사진 변경하기]           │
│                                 │
│  닉네임                          │
│  ┌─────────────────────────┐    │
│  │ 투자왕                   │    │
│  └─────────────────────────┘    │
│  5/20                           │
│                                 │
│  투자 성향                       │
│  ┌────────┐┌────────┐┌────────┐ │
│  │ 안정형 ││[중립형]││ 공격형 │ │
│  └────────┘└────────┘└────────┘ │
│                                 │
│                                 │
│    ┌─────────────────────────┐  │
│    │        저장하기          │  │
│    └─────────────────────────┘  │
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 헤더
| 속성 | 값 |
|------|-----|
| 뒤로가기 | 설정 화면으로 복귀 (변경사항 있으면 확인 다이얼로그) |
| 타이틀 | "프로필 편집", 18px, Bold |

### 2. 프로필 이미지
| 속성 | 값 |
|------|-----|
| 크기 | 100x100 px |
| 형태 | 원형 |
| 편집 | P1에서는 변경 불가 (안내 문구만 표시) |

### 3. 닉네임 입력
| 속성 | 값 |
|------|-----|
| 라벨 | "닉네임" |
| 초기값 | 현재 닉네임 |
| 최대 길이 | 20자 |
| 글자수 표시 | "n/20" (우측 하단) |

### 4. 투자 성향
| 속성 | 값 |
|------|-----|
| 라벨 | "투자 성향" |
| 옵션 | 안정형 / 중립형 / 공격형 |
| 형태 | 토글 버튼 그룹 (하나만 선택 가능) |
| 초기값 | 현재 설정값 |

### 5. 저장하기 버튼
| 속성 | 값 |
|------|-----|
| 너비 | 화면 90% |
| 높이 | 48px |
| 스타일 | Primary 버튼 |
| 비활성 조건 | 변경사항 없음 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 닉네임 수정 | 글자수 업데이트, 저장 버튼 활성화 |
| 성향 변경 | 저장 버튼 활성화 |
| 저장하기 클릭 | API 호출 → 성공 시 이전 화면 복귀 |
| 뒤로가기 | 변경사항 있으면 "저장하지 않고 나가시겠습니까?" 다이얼로그 |

## 상태

| 상태 | UI |
|------|-----|
| 기본 | 버튼 비활성화 |
| 변경됨 | 버튼 활성화 |
| 로딩 중 | 버튼에 스피너, 입력 비활성화 |
| 성공 | 토스트 "저장되었습니다" → 이전 화면 |
| 에러 | 토스트 메시지 |
