# 설정 메인 화면 (Settings)

## 개요

사용자 설정을 관리하는 메인 허브 화면

## 컴포넌트

### 1. 헤더
| 속성 | 값 |
|------|-----|
| 뒤로가기 | 이전 화면으로 복귀 |
| 타이틀 | "설정", 18px, Bold |

### 2. 프로필 카드
| 속성 | 값 |
|------|-----|
| 이미지 | 프로필 사진 (50x50, 원형) |
| 닉네임 | 사용자 닉네임 |
| 이메일 | 사용자 이메일 |
| 탭 액션 | 프로필 편집 화면 이동 |

### 3. 일반 설정 그룹
| 항목 | 설명 | 액션 |
|------|------|------|
| 알림 설정 | 알림 관련 세부 설정 | 알림 설정 화면 이동 |
| 투자 성향 | 현재 성향 표시 | 프로필 편집 화면 이동 |

### 4. 앱 정보 그룹
| 항목 | 설명 | 액션 |
|------|------|------|
| 이용약관 | 서비스 약관 | 웹뷰 열기 |
| 개인정보처리방침 | 개인정보 정책 | 웹뷰 열기 |
| 앱 버전 | 현재 버전 표시 | 없음 |

### 5. 계정 관리
| 항목 | 스타일 | 액션 |
|------|--------|------|
| 로그아웃 | Secondary 버튼 | 로그아웃 확인 다이얼로그 |
| 회원탈퇴 | Danger 텍스트 | 탈퇴 확인 플로우 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 프로필 카드 탭 | 프로필 편집 화면 이동 |
| 알림 설정 탭 | 알림 설정 화면 이동 |
| 투자 성향 탭 | 프로필 편집 화면 이동 |
| 로그아웃 | "로그아웃하시겠습니까?" → 확인 시 로그인 화면 |
| 회원탈퇴 | 탈퇴 사유 선택 → 최종 확인 → 완료 |

## 상태

| 상태 | UI |
|------|-----|
| 로그아웃 확인 | 다이얼로그 표시 |
| 로그아웃 처리 중 | 버튼 로딩 |
| 탈퇴 확인 | 다단계 확인 플로우 |
