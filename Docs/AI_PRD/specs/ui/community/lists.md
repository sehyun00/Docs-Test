# 참조 목록 화면 (Reference Lists Screen)

## 개요

북마크 목록, 차단 계정 관리

---

## 1. 북마크 목록

**진입**: 커뮤니티 설정 > 북마크

```
┌─────────────────────────────────────┐
│  ← 북마크                           │
├─────────────────────────────────────┤
│                                     │
│ ┌─────────────────────────────────┐ │
│ │ [프사] 투자왕김철수              │ │  ← 최신 북마크 순
│ │       3분 전                    │ │
│ │ 삼성전자, 지금 사도 될까요?      │ │
│ │ ❤️ 42  💬 12  🔄 5  📤    ⋯   │ │
│ └─────────────────────────────────┘ │
│ ─────────────────────────────────── │
│ ┌─────────────────────────────────┐ │
│ │ [프사] 배당러버                 │ │
│ │       ...                       │ │
│ └─────────────────────────────────┘ │
│                                     │
│         📚 마지막 글이에요          │
│                                     │
└─────────────────────────────────────┘
```

### 빈 상태

```
┌─────────────────────────────────────┐
│  ← 북마크                           │
├─────────────────────────────────────┤
│                                     │
│            🔖                       │
│     북마크한 글이 없어요            │
│                                     │
│  마음에 드는 글을 북마크해보세요    │
│                                     │
└─────────────────────────────────────┘
```

---

## 2. 차단 계정 관리

**진입**: 커뮤니티 설정 > 차단 계정 관리

```
┌─────────────────────────────────────┐
│  ← 차단한 계정                      │
├─────────────────────────────────────┤
│                                     │
│  [프사] 스팸유저123      [차단해제] │
│  ─────────────────────────────────  │
│  [프사] 악성댓글러       [차단해제] │
│  ─────────────────────────────────  │
│  [프사] 광고봇001        [차단해제] │
│                                     │
└─────────────────────────────────────┘
```

### 빈 상태

```
┌─────────────────────────────────────┐
│  ← 차단한 계정                      │
├─────────────────────────────────────┤
│                                     │
│            🚫                       │
│     차단한 계정이 없어요            │
│                                     │
└─────────────────────────────────────┘
```

## 상호작용

| 액션 | 동작 |
|------|------|
| 북마크 카드 탭 | 게시글 상세 이동 |
| 차단해제 탭 | 확인 모달 후 해제 |
