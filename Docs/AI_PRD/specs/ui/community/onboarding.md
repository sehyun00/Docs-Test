# 온보딩 화면 (Onboarding Screen)

## 개요

커뮤니티 최초 진입 시 약관 동의 및 닉네임 설정

## 레이아웃

```
┌─────────────────────────────────────┐
│              ← 닫기                 │
├─────────────────────────────────────┤
│                                     │
│         🎉 커뮤니티 시작하기         │
│                                     │
│  다른 투자자들과 의견을 나누고       │
│  포트폴리오를 공유해보세요           │
│                                     │
├─────────────────────────────────────┤
│  ☑️ 커뮤니티 이용약관 동의 (필수)    │
│  ☑️ 개인정보 처리방침 동의 (필수)    │
├─────────────────────────────────────┤
│                                     │
│  닉네임 입력                        │
│  ┌─────────────────────────────────┐│
│  │ 투자왕김철수                     ││
│  └─────────────────────────────────┘│
│  ⚠️ 닉네임은 90일마다 변경 가능합니다│
│                                     │
│         [시작하기]                  │
│                                     │
└─────────────────────────────────────┘
```

## 컴포넌트

### 1. 약관 동의

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 커뮤니티 이용약관 | Y | 탭 시 전문 보기 |
| 개인정보 처리방침 | Y | 탭 시 전문 보기 |

### 2. 닉네임 입력

| 요소 | 설명 |
|------|------|
| 입력 제한 | 한글/영어/숫자, 최대 20자 |
| 중복 체크 | 실시간 검증 |
| 안내 문구 | 90일 쿨타임 안내 |

### 3. 시작하기 버튼

- 약관 전체 동의 + 닉네임 유효 시 활성화
- 탭 시 프로필 생성 후 피드 화면 이동

## 비즈니스 규칙

- 모든 약관 동의 필수
- 닉네임 중복 불가
- 닉네임 변경은 90일에 1회만 가능
