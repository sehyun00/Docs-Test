# 프로필 화면 (Profile Screen)

## 개요

내 프로필 / 타인 프로필 / 프로필 편집

---

## 1. 내 프로필

```
┌─────────────────────────────────────┐
│  ←                    ⚙️ (설정)    │
├─────────────────────────────────────┤
│                                     │
│  투자왕김철수              [프사]   │
│  @investking                        │
│                                     │
│  게시글 12  │ 팔로워 156  │ 팔로잉 23│
│                                     │
│  [프로필 편집]                      │
│                                     │
├─────────────────────────────────────┤
│  [최근활동]  [투자]  [TBD]          │
├─────────────────────────────────────┤
│  (해당 탭 콘텐츠)                   │
│                                     │
└─────────────────────────────────────┘
```

### 컴포넌트

| 요소 | 설명 |
|------|------|
| 설정 (⚙️) | 커뮤니티 설정 화면 이동 |
| 닉네임 | 내 닉네임 |
| 통계 | 게시글 / 팔로워 / 팔로잉 수 |
| 프로필 편집 | 편집 화면 이동 |

---

## 2. 타인 프로필

```
┌─────────────────────────────────────┐
│  ←                          [🚫]   │  ← 차단
├─────────────────────────────────────┤
│                                     │
│  배당러버                   [프사]  │
│  @dividend_lover                    │
│                                     │
│  게시글 48  │ 팔로워 1.2K │ 팔로잉 89│
│                                     │
│  [팔로우]                           │
│                                     │
├─────────────────────────────────────┤
│  (공개 설정된 탭만 표시)            │
│                                     │
└─────────────────────────────────────┘
```

### 차이점

| 항목 | 내 프로필 | 타인 프로필 |
|------|:--------:|:----------:|
| 헤더 우측 | ⚙️ 설정 | 🚫 차단 |
| 버튼 | 프로필 편집 | 팔로우 |
| 탭 | 전체 표시 | 공개 탭만 |

---

## 3. 프로필 편집

```
┌─────────────────────────────────────┐
│  ← 취소                    [저장]   │
├─────────────────────────────────────┤
│                                     │
│            ┌───────┐                │
│            │ [프사] │                │
│            └───────┘                │
│         📷 사진 변경                │
│                                     │
├─────────────────────────────────────┤
│  닉네임                             │
│  ┌─────────────────────────────────┐│
│  │ 투자왕김철수                     ││
│  └─────────────────────────────────┘│
│  ⚠️ 닉네임은 90일마다 변경 가능합니다│
│  마지막 변경: 2025-12-01           │
│                                     │
├─────────────────────────────────────┤
│  자기소개                           │
│  ┌─────────────────────────────────┐│
│  │ 장기투자를 지향합니다.           ││
│  └─────────────────────────────────┘│
│  0/200자                            │
│                                     │
└─────────────────────────────────────┘
```

### 입력 제한

| 필드 | 제한 |
|------|------|
| 닉네임 | 한글/영어/숫자, 최대 20자 |
| 자기소개 | 최대 200자 |
| 닉네임 쿨타임 | 90일 |

### 나가기 확인 모달

변경사항 있을 때 취소 클릭 시:
- "변경사항이 저장되지 않아요. 정말 나가시겠어요?"
- [닫기] / [나가기]
