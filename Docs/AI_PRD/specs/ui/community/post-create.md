# 게시글 작성/수정 화면 (Post Create/Edit Screen)

## 개요

게시글 작성, 수정, 종목 태그, 이미지 첨부

## 레이아웃

```
┌─────────────────────────────────────┐
│  ← 취소                    [게시]   │  ← 헤더
├─────────────────────────────────────┤
│                                     │
│  카테고리 선택                      │
│  [전체] [국내] [해외]               │  ← 토글 버튼
│                                     │
├─────────────────────────────────────┤
│                                     │
│  제목을 입력하세요                  │  ← 플레이스홀더
│  ─────────────────────────────────  │
│                                     │
│  내용을 입력하세요.                 │  ← 플레이스홀더
│  커뮤니티 이용규칙에 맞지 않는 글은  │
│  통보 없이 삭제될 수 있습니다.       │
│                                     │
│  🖼️ (첨부 이미지 미리보기)          │
│                                     │
├─────────────────────────────────────┤
│ [삼성전자] [삼성SDI] ...            │  ← 종목 자동완성
├─────────────────────────────────────┤
│  [📷 사진]  [GIF]  [📈 종목태그]    │  ← 키보드 상단 패널
├─────────────────────────────────────┤
│          ⌨️ 키보드                  │
└─────────────────────────────────────┘
```

## 컴포넌트

### 1. 헤더

| 요소 | 설명 |
|------|------|
| ← 취소 | 나가기 (작성 중이면 확인 모달) |
| [게시] | 조건 충족 시 활성화 |

### 2. 게시 버튼 활성화 조건

- 제목 1글자 이상 **AND**
- 본문 1글자 이상

### 3. 카테고리 선택

| 요소 | 설명 |
|------|------|
| 기본값 | 전체 |
| 옵션 | 전체 / 국내 / 해외 |
| UI | 토글 버튼 (단일 선택) |

### 4. 키보드 상단 패널

| 버튼 | 설명 |
|------|------|
| 📷 사진 | 갤러리에서 이미지 선택 |
| GIF | Giphy API 연동 (추후) |
| 📈 종목태그 | `#` 자동 입력 → 종목 검색 |

### 5. 종목 자동완성

- `#` 입력 시 패널에 종목 리스트 표시
- 선택 시 `#삼성전자` 형태로 완성
- 완성된 태그는 하이라이트 (파란색)

## 수정 화면 차이점

| 항목 | 작성 | 수정 |
|------|------|------|
| 헤더 버튼 | [게시] | [수정] |
| 초기값 | 빈 값 | 기존 내용 로드 |
| 완료 후 | 피드 이동 | 상세로 복귀 + "(수정됨)" |

## 나가기 확인 모달

작성 중(입력값 있을 때) 취소 클릭 시:
- "작성 중인 글이 저장되지 않아요. 정말 나가시겠어요?"
- [닫기] / [나가기]
