# 커뮤니티 설정 화면 (Settings Screen)

## 개요

알림 설정, 공개 범위, 기타 커뮤니티 설정

---

## 1. 설정 메뉴

**진입**: 내 프로필 > ⚙️ 설정

```
┌─────────────────────────────────────┐
│  ← 커뮤니티 설정                    │
├─────────────────────────────────────┤
│                                     │
│  > 알림 설정                        │
│  > 정보 공개 범위                   │
│  > 북마크                           │
│  > 차단 계정 관리                   │
│  > 신고·삭제된 글                   │
│  > 커뮤니티 이용규칙                │
│                                     │
└─────────────────────────────────────┘
```

---

## 2. 알림 설정

**진입**: 커뮤니티 설정 > 알림 설정

```
┌─────────────────────────────────────┐
│  ← 알림 설정                        │
├─────────────────────────────────────┤
│                                     │
│  게시글 및 댓글                     │
│  ─────────────────────────────────  │
│  내 글에 좋아요              [━━●]  │
│  내 글에 댓글                [━━●]  │
│  내 글 리포스트              [━━●]  │
│                                     │
│  팔로우                             │
│  ─────────────────────────────────  │
│  새 팔로워                   [━━●]  │
│                                     │
│  내가 팔로우하는 사람의 활동         │
│  ─────────────────────────────────  │
│  새 글                       [━━●]  │
│  거래                        [━━●]  │
│                                     │
└─────────────────────────────────────┘
```

### 알림 항목

| 섹션 | 항목 | 기본값 |
|------|------|:------:|
| **게시글 및 댓글** | 내 글에 좋아요 | ON |
| | 내 글에 댓글 | ON |
| | 내 글 리포스트 | ON |
| **팔로우** | 새 팔로워 | ON |
| **팔로우 활동** | 새 글 | ON |
| | 거래 | ON |

---

## 3. 정보 공개 범위

```
┌─────────────────────────────────────┐
│  ← 정보 공개 범위                   │
├─────────────────────────────────────┤
│                                     │
│  최근 활동                   [━━●]  │
│  투자 현황                   [●━━]  │ ← P3 (기본 OFF)
│  거래 내역                   [━━●]  │
│                                     │
└─────────────────────────────────────┘
```

### 공개 설정

| 항목 | 기본값 | Phase |
|------|:------:|:-----:|
| 최근 활동 | ON | P2 |
| 투자 현황 | OFF | P3 |
| 거래 내역 | ON | P2 |

## 접근 경로

1. 커뮤니티 > 프로필 > ⚙️ > 알림 설정
2. 앱 전체 설정 > 커뮤니티 (탭 추가 예정)
