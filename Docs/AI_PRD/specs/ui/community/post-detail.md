# 게시글 상세 화면 (Post Detail Screen)

## 개요

게시글 본문, 댓글/대댓글 표시

## 컴포넌트

### 1. 프로필 영역

| 요소 | 설명 |
|------|------|
| 프사 | 클릭 → 프로필 상세 |
| 닉네임 | 작성자 닉네임 |
| 날짜 | n분 전 / m월 d일 (7일 이후) |
| (수정됨) | 수정된 글이면 표시 |
| 팔로워 수 | N / N.NK / N.NM |
| 팔로우 버튼 | 팔로우/팔로잉 토글 |

### 2. 댓글 영역

| 요소 | 설명 |
|------|------|
| 정렬 드롭다운 | 최신순(기본) / 인기순 / 날짜순 |
| 댓글 구성 | 프사 / 닉네임 · 시간 / 내용 |
| ❤️ | 댓글 좋아요 |
| ↩️ | 대댓글 작성 |

### 3. 대댓글 작성

- ↩️ 클릭 시 `@닉네임` 자동 입력
- 일반 댓글로 등록 (스레드 X)

## 상호작용

| 액션 | 동작 |
|------|------|
| 프사 탭 | 프로필 화면 이동 |
| 팔로우 탭 | 팔로우/언팔로우 토글 |
| 좋아요 탭 | 좋아요 토글 |
| 공유 탭 | 공유 시트 표시 |
