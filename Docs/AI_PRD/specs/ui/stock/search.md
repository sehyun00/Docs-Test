# 종목 검색 화면 (Stock Search)

## 개요

종목 검색 및 선택

## 레이아웃

```
┌─────────────────────────────────┐
│  🔍 종목 검색                    │
│  ┌─────────────────────────────┐ │
│  │ 삼성                         │ │
│  └─────────────────────────────┘ │
├─────────────────────────────────┤
│  검색 결과                       │
│  ┌─────────────────────────────┐ │
│  │ 삼성전자 (005930)    ₩75,000 │ │
│  │ KOSPI           +2.32%      │ │
│  └─────────────────────────────┘ │
│  ┌─────────────────────────────┐ │
│  │ 삼성전자우 (005935)  ₩62,000 │ │
│  │ KOSPI           -0.80%      │ │
│  └─────────────────────────────┘ │
│                                  │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 검색 입력창

- 실시간 검색 (debounce 300ms)
- 종목명 또는 종목코드로 검색

### 2. 검색 결과 목록

| 요소 | 설명 | 비고 |
|------|------|------|
| 종목명 | 종목 이름 (코드) | |
| 시장 | KOSPI/KOSDAQ | |
| 현재가 | 현재 가격 (₩/$/¥ 통화 기호) | |
| 등락률 | 양수: 빨간색, 음수: 파란색, 0: 회색 | 색상 구분 |
| 보유 배지 | 이미 포트폴리오에 있는 종목 시 표시 | Dimmed 처리 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 종목 선택 | 종목 추가 모달 표시 |
| 보유 종목 선택 | 탭 비활성화 (Dimmed, 선택 불가) |
| 검색어 입력 | 실시간 검색 결과 표시 |

## 상태

| 상태 | UI |
|------|-----|
| 로딩 중 | 스켈레톤 |
| 검색 결과 없음 | "검색 결과가 없습니다" |
| 에러 | 에러 메시지 + 재시도 버튼 |
