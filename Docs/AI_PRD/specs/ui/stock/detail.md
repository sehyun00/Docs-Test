# 종목 상세 화면 (Stock Detail)

## 개요

개별 종목의 상세 정보, 차트, 내 보유/비율 현황을 한눈에 볼 수 있는 상세 페이지

## 레이아웃

```
┌─────────────────────────────────┐
│  ← 삼성전자                      │
├─────────────────────────────────┤
│                                 │
│  ₩75,000  +2,000 (+2.74%)      │
│                                 │
│  ┌─────────────────────────────┐│
│  │                             ││
│  │      [ 캔들 차트 영역 ]       ││
│  │                             ││
│  └─────────────────────────────┘│
│  [1일] [1주] [1달] [3달]        │
│                                 │
│  ─────────────────────────────  │
│                                 │
│  내 보유 현황                   │
│  ┌─────────────────────────────┐│
│  │ 보유 수량      10주          ││
│  │ 평균 단가      ₩70,000      ││
│  │ 평가 금액      ₩750,000     ││
│  │ 수익률         +7.14%       ││
│  │ 목표 비율      30.0%        ││
│  └─────────────────────────────┘│
│                                 │
│  ┌────────────┐ ┌────────────┐  │
│  │   수정     │ │    삭제    │  │
│  └────────────┘ └────────────┘  │
│                                 │
└─────────────────────────────────┘
```

## 컴포넌트

### 1. 헤더

| 요소 | 설명 |
|------|------|
| 뒤로가기 | 포트폴리오 상세로 복귀 |
| 종목명 | 화면 타이틀 |

### 2. 현재가 섹션

| 요소 | 설명 |
|------|------|
| 현재가 | 실시간 가격 (price API 캐시) |
| 변동폭 | 전일 대비 금액 + 퍼센트 |
| 색상 | 상승: 빨강, 하락: 파랑 |

### 3. 차트 영역

| 요소 | 설명 |
|------|------|
| 캔들 차트 | OHLC 봉차트 (시가/고가/저가/종가) |
| 기간 탭 | 1일(1d), 1주(1w), 1달(1m), 3달(3m) |

### 4. 보유 현황 카드

| 요소 | 설명 |
|------|------|
| 보유 수량 | 현재 보유 주식 수 |
| 평균 단가 | 매수 평균가 |
| 평가 금액 | 현재가 × 보유수량 |
| 수익률 | (현재가 - 평균단가) / 평균단가 × 100 |
| 목표 비율 | 포트폴리오 내 목표 %age |

### 5. 액션 버튼

| 요소 | 설명 |
|------|------|
| 수정 버튼 | 수량/비율 수정 모달 or 화면 이동 |
| 삭제 버튼 | 종목 삭제 확인 다이얼로그 |

## 상호작용

| 액션 | 동작 |
|------|------|
| 기간 탭 클릭 | history API 호출 → 차트 갱신 |
| 수정 버튼 | 종목 수정 화면/모달 표시 |
| 삭제 버튼 | 확인 다이얼로그 → 삭제 API 호출 |
| 뒤로가기 | 포트폴리오 상세 화면으로 복귀 |

## 상태

| 상태 | UI |
|------|-----|
| 차트 로딩 중 | 차트 영역 스켈레톤 |
| 차트 실패 | "차트를 불러올 수 없습니다" |
| 삭제 확인 | 확인/취소 다이얼로그 |
| 삭제 처리 중 | 버튼 로딩 상태 |

## 데이터 소스

| 항목 | 소스 |
|------|------|
| 종목 기본 정보 | 포트폴리오 상세에서 전달받은 state (API 재호출 X) |
| 현재가 | price API (캐시 30초) |
| 차트 데이터 | history API |
