## 💡 핵심 개념 미리보기

### 📊 DB 관계 (1:N)

```
users → portfolios → portfolio_entries
 1명  →  여러 개    →     여러 종목
```

### 🔐 토큰 인증 흐름

```
로그인 → Access Token 발급 → API 호출 시 첨부 → 만료 시 Refresh로 갱신
```

### 📦 상태 분리

| 서버 상태     | 클라이언트 상태  |
| ------------- | ---------------- |
| 사용자 프로필 | 모달 열림 여부   |
| 게시글 목록   | 입력 중인 텍스트 |
| 상품 재고     | 현재 탭          |

### ⚡ 성능 핵심

```
측정 먼저 → 병목 찾기 → 최적화 → 다시 측정
```
