## 🎯 요구사항 요약

### 핵심 기능 영역 (Phase 1)

```
📱 인증 & 계정     → Google OAuth, JWT, 자동 토큰 갱신
📊 포트폴리오 관리  → CRUD, 최대 5개, 실시간 시세
📈 종목 관리       → 검색, 추가/수정/삭제, 소수점 4자리
⚖️ 리밸런싱       → 목표 비율 설정, 매수/매도 제안
🔔 알림           → 리밸런싱 알림, 주기 설정
⚙️ 설정           → 프로필 수정, 약관, 버전 정보
```

---

## 🛠️ 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | React Native (Expo), Context API |
| Backend | Spring Boot 3.x, Java 17, JPA |
| Database | MySQL 8.0 (AWS RDS) |
| 인증 | Google OAuth 2.0, JWT (Access 1h, Refresh 14d) |
| 외부 API | 한국투자증권 API |
| 인프라 | AWS EC2 + RDS, GitHub Actions |

---

## 📊 DB 스키마 요약

```
users (사용자)
├── portfolios (포트폴리오, 최대 5개)
│   ├── stocks (종목)
│   └── notifications (알림 설정)
```

| 테이블 | 핵심 컬럼 |
|--------|----------|
| users | email, google_id, profile_image_url |
| portfolios | name (20자), description (100자), display_order |
| stocks | stock_code, quantity (소수점 4자리), target_ratio |
| notifications | frequency, notification_time, threshold |
